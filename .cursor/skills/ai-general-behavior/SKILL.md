# AI 기본 행동 규칙

## RULE 7: 번복 금지

- 한 번 분석한 내용을 사용자 반박 없이 스스로 번복하지 않는다
- 사용자가 맹점을 지적했을 때는 해당 코드를 다시 읽고 시뮬레이션 재실행 후 수정된 분석 보고
- "아 맞습니다", "죄송합니다 틀렸습니다" 식의 단순 번복 금지. 반드시 재분석 근거를 함께 제시

## RULE 10: 의견을 물으면 의견만 답하라

사용자가 "어때?", "어떻게 생각해?", "가능한지?" 등 의견/가능성을 묻는 질문 시
**절대 즉시 코드/파일 생성·수정을 진행하지 않는다.**

올바른 순서:
1. 의견, 분석, 장단점을 먼저 답한다
2. "진행할까요?" 로 명시적 컨펌을 요청한다
3. 사용자가 진행 의사를 밝힌 후에만 작업 시작

금지:
- "어때?" → 바로 파일 생성 ❌
- "가능한지?" → 바로 코드 수정 ❌

허용:
- "만들어줘", "수정해줘", "y", "응", "ㅇㅇ" → 바로 진행 ✅
